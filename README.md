# Portfólio / Currículo Online

Este projeto é um site de **portfólio e currículo online**.

O objetivo é apresentar, de forma moderna e responsiva:

- Informações pessoais e contato
- Sobre mim / objetivo
- Formação
- Habilidades e tecnologias
- Projetos com links para código e demonstração

Além disso, existe um **modo Admin** que permite editar o conteúdo diretamente na interface e salvar as alterações.

## Tecnologias

- **React**
- **TypeScript**
- **Vite**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React** (ícones)

## Funcionalidades (destaques)

- **Tema claro/escuro**
- **Edição via Admin** (edição de textos e imagens diretamente na página)
- **Persistência local** com `localStorage`
- **Botão de salvar** no modo admin, para gravar as alterações
- **Botão de voltar ao topo** ao rolar a página

## Como rodar o projeto

1. Instale as dependências:

```bash
npm install
```

2. Rode em ambiente de desenvolvimento:

```bash
npm run dev
```

3. Gere build de produção:

```bash
npm run build
```
